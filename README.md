# go-chat

Simple chat 


## Usage dev-mode


Clone the repository:
 git clone https://github.com/YankinA/go-chat.git

To run databases:

```sh
cd server
```

```sh
docker-compose up
```

To run the server:

```sh
cd server
```


```sh
go run main.go
```

or (recomend)
```sh
  fresh main.go
```
 


Run the clinet:
```sh
cd client
```

```sh
npm ci
```

```sh
npm run dev
```
